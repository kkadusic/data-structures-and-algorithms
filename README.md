# Data Structures and Algorithms

Widely used data structures and algorithms implemented in C++.

### Implementation tree

```
Data Structures and Algorithms
├── Deque
│   └── deque.cpp                     # Deque implementation
├── DirectedGraph
│   └── directed-graph.cpp            # Graph implementation using adjacency matrix
├── Heap
│   └── heap.cpp                      # Heap implementation
├── LinkedList
│   ├── doubly-linked-list.cpp        # Doubly linked list implementation
│   └── singly-linked-list.cpp        # Singly linked list implementation
├── Map
│   ├── avl-tree-map.cpp              # Map implementation using AVL tree
│   ├── binary-tree-map.cpp           # Map implementation using BST
│   ├── hash-map.cpp                  # Hash map implementation
│   └── map.cpp                       # Dynamically allocated array map implementation
├── Queue
│   └── queue.cpp                     # Queue implementation
├── Recursion
│   └── recursion.cpp                 # Fibonacci number and GCD using recursion
├── SortingAlgorithms
│   └── sorting-algorithms.cpp        # Bubble, selection, quick and merge sort
└── Stack
    ├── stack-binary-search.cpp       # Binary search implementation using stack
    └── stack.cpp                     # Stack implementation
```
